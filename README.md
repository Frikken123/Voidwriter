# Voidwriter
A distraction-free writing app inspired by the typewriters of old. Admitedly vibecoded.

Welcome to VoidWriter!

VoidWriter is a minimalist writing tool designed to help you focus on your work.

Where are my files?
All your work is saved in the VoidWriter\writings folder located in your user home directory (C:\Users\[YourName]\VoidWriter\writings). You can easily find this by opening your File Explorer and clicking on your username in the sidebar.

How it works

    Saving: Your work is saved automatically while you type.

    Projects: Each session creates a .txt file for your raw text and a .vwproject file for your internal project state.

    Page Snapshots: When you close the application, VoidWriter automatically compiles your writing into high-resolution page images (PNGs) and saves them in a folder named after your document (e.g., new-draft-1_pages).

    Keyboard Controls:

        Escape: Minimize the application.

        Ctrl + Q: Quit the application.

Troubleshooting
If you need to move your files or change the installation location, you can do so through the standard Windows installation settings provided during setup.
